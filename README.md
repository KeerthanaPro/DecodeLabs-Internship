## 📖 Description

A **Rule-Based AI Chatbot Assistant** built with Python Flask that responds to predefined user queries using decision-making logic. This project demonstrates the IPO (Input-Process-Output) model with a modern dashboard UI.

## ✨ Features

### Core Functionality
- ✅ Rule-based response engine using dictionary mapping
- ✅ Input sanitization (lowercase, remove punctuation)
- ✅ Dynamic time response
- ✅ Unknown query tracking with counter
- ✅ Real-time chat interface
- ✅ Analytics dashboard with statistics

### Modules Implemented
- **Module 1:** Greeting Detection (hello, hi, hey, good morning)
- **Module 2:** College Information (RMK Engineering College)
- **Module 3:** Internship Queries (internship, project, training)
- **Module 4:** Time Query (time, current time)
- **Module 5:** Exit Commands (bye, exit, quit)

↓

↓

↓

↓
Chat Window Output
